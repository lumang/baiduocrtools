# baiduocrtools 百度文字识别接口调试工具
baidu ocr api tools 
百度 ocr 文字识别接口调试工具
~~[参考教程 [只要10分钟 快速掌握文字识别 教程]](https://ai.baidu.com/forum/topic/show/867951)~~


这个教程对于开发人来说是比较简单的
但是考虑到很多非开发人员也会用到百度文字识别的功能，
postman 软件非开发人员显得有点复杂，难懂，不好操作
这里提供一个百度文字识别的工具，希望可以对你有帮助
